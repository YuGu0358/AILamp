# 5. 控制、视觉与仿真

仿真：

```bash
ailamp sim-demo
ailamp sim-viewer --render outputs/ailamp.png
```

真实硬件：

```bash
ailamp camera-test
ailamp audio-test
ailamp led-test
ailamp vision-demo
ailamp agent
```

事件映射：

```text
no_person -> scanning -> 深蓝
person_left -> look_left -> 蓝色
person_center -> nod -> 暖色
person_right -> look_right -> 蓝色
person_close -> shy -> 粉红
person_far -> curious -> 绿色
```
