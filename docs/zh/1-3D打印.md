# 1. 3D 打印

使用 `3D/` 目录中的 LeLamp 原始 `.3mf` 文件。

## CAD 建模来源

AILamp 保持原 LeLamp 的机械建模流程：

- 主 CAD 软件：OnShape。
- 原始 CAD 链接：`https://cad.onshape.com/documents/16c9706360b5ad34f9c8db49/w/2edfa54c83253c120fbc9e58/e/a7196194821d9cfe2842a44a`
- 原始装配链接：`https://cad.onshape.com/documents/16c9706360b5ad34f9c8db49/w/2edfa54c83253c120fbc9e58/e/a35eec618cd78ea5d74bf01b`
- 3D 打印导出格式：`.3mf`。
- 仿真网格导出格式：`.stl`。

AILamp v1 不把 Blender、SolidWorks、Fusion 360、FreeCAD 作为主建模流程。

需要打印：

```text
LampBase.3mf x1
LampBase - Cover.3mf x1
LampArm (Base-Elbow).3mf x1
LampArm (Elbow-Wrist).3mf x1
LampArm (Pitch).3mf x2
LampHead.3mf x1
LampHead - Diffuser.3mf x1
```

## AILamp 替换底座套件

AILamp v1 保留 LeLamp 原始 7 个 `.3mf` 文件作为参考。Jetson Nano 隐藏电子件版本不要把原始 `LampBase.3mf` 和 `LampBase - Cover.3mf` 作为最终打印件，而是打印 AILamp 替换底座壳体和替换上盖。

尺寸校核：当前按 Jetson Nano 4GB 开发套件 100 x 80 x 29 mm、Raspberry Pi Pico WH 51 x 21 mm、Waveshare Servo Driver with ESP32 65 x 30 mm 建模。原 LeLamp `LampBase.3mf` 外形约为 160 x 190 x 40 mm，所以 AILamp 替换底座扩大到 190 x 230 x 48 mm，并把电子件放进底座本体。

外形构建方式：本版本不再是把原 `LampBase.3mf` 网格拉伸成新尺寸，而是直接用圆角矩形多边形+三角化挤出生成壳体和上盖。所有外圆角和上盖外周的过渡都是真正连续的圆弧（每角 24 段），不再是从盒子里减条带模拟出来的阶梯。下沉 lip、灯臂根部 collar 和过渡靴 boot 也按同一规则做。结果是打印件外形显著更平滑、文件体积也下降一半以上（壳体 .stl 从约 2.5 MB 缩到约 0.9 MB，上盖从约 8.5 MB 缩到约 0.6 MB）。

| 适配件文件 | 数量 | 用途 |
| --- | ---: | --- |
| `AILamp_LampBase_Electronics_Shell.3mf` | 1 | 替换版 LampBase 壳体，容纳 Jetson Nano、Pico WH、ST3215 驱动板、走线和散热空间 |
| `AILamp_LampBase_Electronics_Cover.3mf` | 1 | 替换版 LampBase 上盖，带凸起灯臂固定座、固定孔和检修通风孔 |
| `AILamp_Base_Arm_Link_Boot.3mf` | 1 | 固定上盖与运动灯臂根部之间的随动过渡靴 |
| `AILamp_Cable_Clip_6mm.3mf` | 2-4 | USB 和信号线走线 |
| `AILamp_Cable_Clip_10mm.3mf` | 2-4 | 电源线和舵机线走线 |
| `AILamp_Jetson_Nano_Base_Tray.3mf` | 可选 | 台架调试用 Jetson 外置托盘 |
| `AILamp_Electronics_Side_Deck.3mf` | 可选 | 台架调试用 Pico WH 和 ST3215 驱动板侧板 |

每个适配件 `.3mf` 都有同名 `.stl` 导出文件，用于切片软件兼容和外形检查。

替换底座几何关系：

- 替换壳体外形：190 x 230 x 48 mm。
- 替换上盖板体：190 x 230 x 8 mm，包含凸起底座-灯臂固定座后的总高度为 22 mm。
- 下层检修通过孔：72 x 86 mm。
- 凸起灯臂固定座：92 x 98 x 18 mm，中间给底座舵机/灯臂根部预留 58 x 64 mm 宽松避让。
- 灯臂固定座孔位：x 方向 +/-36 mm，y 方向 +/-40 mm。
- 随动底座-灯臂过渡靴：74 x 74 x 42 mm，中间预留 42 x 48 mm 宽松避让。MuJoCo 中这个过渡靴挂在原 LeLamp 上盖 transform 上，会跟随灯臂根部，而不是跟固定电子仓壳体绑定。
- 原底座参考 footprint：160 x 190 mm，用于放大替换壳体比例并保留原本前方更长的底座布局。
- 上盖与壳体固定孔位：x 方向 +/-82 mm，y 方向 -88 mm 和 +118 mm。Y 方向非对称是为了保留原 LeLamp 底座相对灯臂原点的前后关系。
- Jetson 内部定位柱：84 x 64 mm 孔位，3.4 mm 宽松螺丝间隙。
- Waveshare 驱动板定位柱：58 x 23 mm 孔位，对应驱动板固定孔距。

当前替换底座版本不要打印或安装灯头适配件。摄像头、灯板和音频硬件先按线缆/设备接入处理；它们的机械安装需要等 Jetson Nano 底座布局确认后，再单独做适配。

尺寸规则：第一版适配件不要卡太死。PCB 口袋每边预留 1.5 mm，线缆出口至少额外预留 2.5 mm，固定优先使用螺丝、扎带或可拆绑带，不依赖硬卡扣挤压板子。

隐藏电子件路线优先低填充打印 `AILamp_LampBase_Electronics_Shell.3mf`、`AILamp_LampBase_Electronics_Cover.3mf` 和 `AILamp_Base_Arm_Link_Boot.3mf` 做试装。它们在 AILamp 装配中替代原 LeLamp 底座和上盖。
