# 0. 前期准备

## 固定硬件清单

硬件型号和数量以 `config/hardware.toml` 为准，不能随意替换。

| 子系统 | 型号 | 数量 |
| --- | --- | ---: |
| 主控 | NVIDIA Jetson Orin Nano Super Developer Kit, MPN `945-13766-0000-000` | 1 |
| 存储 | Samsung 980 NVMe M.2 2280 500GB, `MZ-V8V500B` | 1 |
| 舵机 | Waveshare ST3215 Servo, SKU `22414` | 5 |
| 舵机驱动 | Waveshare Servo Driver with ESP32, SKU `21593` | 1 |
| LED 控制 | Raspberry Pi Pico WH | 1 |
| LED 面板 | Adafruit NeoPixel NeoMatrix 8x8, Product ID `1487` | 1 |
| 摄像头 | Arducam `UB0234` USB UVC Camera | 1 |
| 音频 | Seeed Studio ReSpeaker XVF3800 USB 4-Mic Array, Product `6490` | 1 |

## 默认端口

```text
/dev/ttyACM0 -> Waveshare 舵机驱动板
/dev/ttyACM1 -> Raspberry Pi Pico WH
/dev/video0  -> Arducam UB0234
```
