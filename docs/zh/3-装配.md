# 3. 装配

连接：

```text
Jetson USB -> Waveshare Servo Driver with ESP32 -> 5x ST3215
Jetson USB -> Raspberry Pi Pico WH -> TXS0108E -> NeoMatrix DIN
Jetson USB -> Arducam UB0234
Jetson USB -> ReSpeaker XVF3800
GST120A12-P1J -> 舵机供电
GST60A05-P1J -> LED 供电
```

首次上电顺序：

1. 只开 Jetson。
2. 测试摄像头和音频。
3. 刷 Pico WH 固件。
4. 测试 LED。
5. 打开舵机电源。
6. 校准舵机。
7. 运行整机 demo。
