# 3. 装配

## 机械适配件安装

AILamp v1 保留 LeLamp 原始打印件不切割。新硬件通过 `3D/AILamp_Adapters/` 中的可逆适配件安装。

1. 先完成 LeLamp 原始底座、灯臂、灯头和扩散片装配。
2. 将 `AILamp_Jetson_Nano_Base_Tray.3mf` 固定在底座旁边或后方，给 Jetson Nano 4GB 保留底部通风空间。
3. 将 `AILamp_Electronics_Side_Deck.3mf` 固定在底座附近，保证 USB、舵机总线、12V 输入和 5V LED 线都能插拔。
4. 将 `AILamp_Head_Camera_Mount.3mf` 作为可逆支架装到灯头上。第一版不要切割 `LampHead.3mf`。
5. 将 `AILamp_NeoMatrix_Holder.3mf` 安装在扩散片后方，保留小间隙，避免 LED 直接顶住扩散片。
6. 将 `AILamp_ReSpeaker_External_Mount.3mf` 放在底座上方或后侧，不遮挡麦克风开孔和 USB 接口，并远离舵机驱动板。
7. `AILamp_Cable_Clip_6mm.3mf` 用于 USB/信号线，`AILamp_Cable_Clip_10mm.3mf` 用于电源线或舵机线束。

尺寸规则：第一版适配件宁可稍松，不要卡死。确认板子能装入后，再用螺丝、扎带或可拆绑带固定。

连接：

```text
Jetson USB -> Waveshare Servo Driver with ESP32 -> 5x ST3215
Jetson USB -> Raspberry Pi Pico WH -> TXS0108E -> NeoMatrix DIN
Jetson USB -> Arducam UB0234
Jetson USB -> ReSpeaker XVF3800
GST120A12-P1J -> 急停开关 -> DC barrel terminal -> WAGO 221-413 -> 舵机驱动板供电
GST60A05-P1J -> DC barrel terminal -> WAGO 221-413 -> NeoMatrix 5V/GND
需要信号连接的设备按接口要求共地
```

LED 保护接线：

```text
Pico GP0 -> TXS0108E A 侧输入
Pico 3V3 -> TXS0108E VCCA
NeoMatrix 5V -> TXS0108E VCCB
公共 GND -> TXS0108E GND
TXS0108E B 侧输出 -> 330 ohm resistor -> NeoMatrix DIN
TXS0108E OE -> VCCA
GST60A05-P1J 5V/GND -> 1000 uF capacitor -> NeoMatrix 5V/GND
```

首次上电顺序：

1. 只给 Jetson 上电。
2. 运行 `ailamp hardware-check`。
3. 接入 Arducam 和 ReSpeaker，然后运行 camera/audio 测试。
4. 刷 Pico WH 固件，并运行 `ailamp led-test`。
5. 给 ST3215 舵机链上电，并运行舵机校准。
6. 运行 `ailamp vision-demo`。
7. 运行 `ailamp agent`。
