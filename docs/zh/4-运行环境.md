# 4. 运行环境

```bash
cd ~/projects/AILamp
python3 -m venv .venv
source .venv/bin/activate
pip install -e ".[hardware,simulation,voice,test]"
```

Jetson Nano 4GB API-hybrid 安装：

```bash
cd ~/projects/AILamp
python3.11 -m venv .venv
source .venv/bin/activate
pip install -e ".[nano,test]"
export OPENAI_API_KEY=...
```

Jetson Nano 配置下不要在 Nano 上安装本地 YOLO pose、MuJoCo 或本地大模型。

检测：

```bash
ailamp hardware-check
ailamp hardware-check --include-devices
ailamp --config config/hardware.jetson-nano.toml hardware-check
ailamp --config config/hardware.jetson-nano.toml hardware-check --include-devices
```

Pico WH 固件在：

```text
firmware/pico_led_controller/code.py
```

Pico WH 要求：

```text
Raspberry Pi Pico WH 使用 CircuitPython 9.x
/lib 中放入 adafruit-circuitpython-neopixel 库
NeoMatrix 数据线接 Pico GP0
Pico GP0 与 NeoMatrix DIN 之间使用 TXS0108E 电平转换
```

串口协议：

```text
PING
CLEAR
SOLID r g b
BRIGHTNESS value
PIXELS r,g,b;r,g,b
```
