# 6. 常见问题

## 设备端口变化

```bash
ls /dev/ttyACM* /dev/ttyUSB* /dev/video*
```

然后修改 `config/hardware.toml`。

Jetson Nano 使用 `config/hardware.jetson-nano.toml`，修改后运行：

```bash
ailamp --config config/hardware.jetson-nano.toml runtime-check --include-devices
```

## Runtime Check 失败

- 缺少 `OPENAI_API_KEY`：Nano API-hybrid 视觉运行前先导出环境变量。
- 缺少 LiveKit/OpenAI 依赖：重新运行 `pip install -e ".[nano]"`。
- 缺少 LeLamp motor runtime：真实 ST3215 回放前安装相邻目录中的 `lelamp_runtime`。
- `outputs/` 不可写：先修复项目目录权限，再运行 `vision-loop`。

## LED 不亮

- 检查 `GST60A05-P1J`。
- 检查 Pico WH。
- 检查 TXS0108E。
- 检查 330 ohm 电阻。
- 运行 `ailamp led-test`。

## 舵机不动

- 检查 `GST120A12-P1J`。
- 检查 Waveshare 驱动板。
- 检查 ST3215 ID。
- 运行 `ailamp motor-test`。

## 仿真检查失败

- 打开 viewer 前先运行 `ailamp sim-check`。
- 如果 actuator 映射失败，先检查 `simulation/robot.xml`，不要直接回放动作。
- 如果渲染失败，安装仿真依赖：`pip install -e ".[simulation]"`。
