# 6. 常见问题

## 设备端口变化

```bash
ls /dev/ttyACM* /dev/ttyUSB* /dev/video*
```

然后修改 `config/hardware.toml`。

## LED 不亮

- 检查 `GST60A05-P1J`。
- 检查 Pico WH。
- 检查 TXS0108E。
- 检查 330 ohm 电阻。
- 运行 `ailamp led-test`。

## 舵机不动

- 检查 `GST120A12-P1J`。
- 检查 Waveshare 驱动板。
- 检查 ST3215 ID。
- 运行 `ailamp motor-test`。
